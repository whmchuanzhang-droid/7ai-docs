# 📦 ChatBox 接入指南

ChatBox 适合新手，是一款跨平台客户端，配置入口极其清楚。

### 1. 下载安装
进入 ChatBox 官网下载对应版本。安装完成后打开软件，不需要登录账号也可以配置咱们的 API。

### 2. 创建 7AI 的 API Key
打开 7AI 个人中心 → API Key 管理 → 新建。名称写 `ChatBox`，配额建议先设一个小额度测试。复制生成的 `sk-` 开头的密钥。

### 3. 进入 ChatBox 设置
点击左下角设置图标，找到模型服务。服务商选择 **OpenAI API** 或 **自定义 OpenAI**。

### 4. 填写 7AI 参数
* **API Key:** 粘贴你的 7AI API Key
* **API Host / Base URL:** `https://api.7ai.one/v1`
* **Model:** 填入你想用的模型（如 `plus-gpt-5.5`）

> ⚠️ **注意：** 如果界面有“自定义模型”，必须手动新增完整名字。不要只填 gpt-5.5，前面的前缀必须保留！

### 5. 保存并测试
新建聊天，输入“你好，测试一下”。能正常回复就大功告成了！如果提示 401 或 Invalid token，说明你的 Key 没复制完整。
