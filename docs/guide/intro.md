# 从网页聊天到本地部署，一页讲清楚

普通用户先用网页聊天；要接入本地软件、脚本或项目时，再创建 API Key。7AI 提供统一余额、网页聊天、CDK 充值和 OpenAI 兼容接口。

## 🚀 快速开始

### 1. 注册账号
进入 7AI 控制台，点击登录/注册。新用户赠送体验额度，可以先试用模型是否适合自己。

### 2. 先用网页聊天
顶部点击“网页聊天”，选择所需分组，再选择具体模型开始对话或生成图片。

### 3. 额度不够再兑换
获取兑换码后，回到个人中心的钱包页面兑换，额度会进入你的账号余额。

### 4. 需要外部工具时再创建 API Key
ChatBox、Cherry Studio、代码项目都需要 API Key；纯网页聊天不需要你手动复制密钥。

---

## 💡 三个核心概念

* **🟣 接口入口 (Base URL)**
  本地软件或代码项目要填的服务地址。OpenAI 兼容工具一般填写：
  `https://api.7ai.one/v1`

* **🟢 调用密钥 (API Key)**
  相当于本地部署软件和代码项目的“登录凭证”。谁拿到你的 Key，谁就可能消耗你的余额，所以千万别发给别人。

* **🟡 额度兑换码 (CDK)**
  拿到兑换码后，回到 7AI 钱包页面兑换。兑换后所有模型共用一个余额。

| 分组 | 计费倍率 | 适用场景 |
| :--- | :--- | :--- |
| **GPT-Plus** | 1x | 日常默认。适合普通聊天、写文案、翻译。建议新手先从这里开始。 |

| **Gemini** | 1x | Gemini 全家桶。适合长上下文、多模态理解和快速分析。 |
| **Claude Anti** | 5x | Claude 高质量通道。适合长文档、代码和深度写作。 |

> **统一余额说明：** 兑换码兑换到账号余额后，网页聊天和本地 API 调用共用这份余额，系统会按对应倍率扣费。
